# linux-ops-toolkit
Collection of Linux operational automation scripts for monitoring, storage management, process health checks, and server maintenance inspired by real-world production support and DevOps activities.

## Features

- Log cleanup automation
- Disk usage monitoring
- Heap dump maintenance
- Process health checks
- Port validation
- Service account rotation workflow
- Scheduled automation examples

## Use Cases

These scripts are based on common operational tasks such as:

- Preventing disk space issues
- Maintaining Linux server health
- Monitoring critical services
- Supporting production environments

## Technologies

- Linux
- Bash
- Cron
